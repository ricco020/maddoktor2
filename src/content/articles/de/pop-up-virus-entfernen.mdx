Endlose Pop-ups – gefälschte Viruswarnungen, „Ihr PC ist infiziert"-Meldungen, zwielichtige Werbung oder ein Browser, der ständig neue Tabs öffnet – werden fast immer durch **Adware** oder **Browser-Benachrichtigungs-Spam** verursacht, nicht durch einen klassischen dateiinfizierenden Virus. Die gute Nachricht ist, dass „Pop-up-Viren" meist zu den am einfachsten zu bereinigenden Infektionen gehören, sobald man weiß, wo sie sich verstecken. So stoppen Sie sie endgültig.

Zuerst eine wichtige Warnung: Die Pop-ups selbst *sind* oft der Betrug. Eine Seite, die „RUFEN SIE DIESE NUMMER AN" oder „Microsoft hat einen Virus erkannt, laden Sie dieses Tool herunter" schreit, ist ein Tech-Support-Betrug. **Rufen Sie nie die Nummer an, geben Sie nie Kartendaten ein und installieren Sie nie die angebotene „Lösung".** Schließen Sie es einfach und folgen Sie den Schritten unten.

## Schritt 1: Schließen Sie das Pop-up sicher

Wenn sich ein Pop-up nicht schließen lässt oder den Bildschirm füllt:

- Klicken Sie nichts darin an – nicht einmal das „X", wenn es gefälscht aussieht, da das eine getarnte Schaltfläche sein kann.
- Öffnen Sie den **Task-Manager** (Strg + Umschalt + Esc), wählen Sie Ihren Browser und klicken Sie auf **Task beenden**, um ihn zwangsweise zu schließen.
- Wenn Sie den Browser wieder öffnen, **stellen Sie die vorherigen Tabs nicht wieder her**, falls er das anbietet – das würde die bösartige Seite neu laden.

## Schritt 2: Entziehen Sie Benachrichtigungsberechtigungen

Eine sehr häufige Ursache für „Pop-ups, sogar wenn der Browser geschlossen ist" ist **Website-Benachrichtigungs-Spam**: Irgendwann haben Sie bei der Benachrichtigungsaufforderung einer Seite auf „Zulassen" geklickt, und jetzt schiebt sie Werbung. Entfernen Sie sie:

- **Chrome:** Einstellungen → Datenschutz und Sicherheit → Website-Einstellungen → Benachrichtigungen. Entfernen Sie jede Seite, die Sie nicht kennen.
- **Edge:** Einstellungen → Cookies und Websiteberechtigungen → Benachrichtigungen. Entfernen Sie unbekannte Seiten.
- **Firefox:** Einstellungen → Datenschutz & Sicherheit → Berechtigungen → Benachrichtigungen → Einstellungen. Entfernen Sie unbekannte Seiten.

Dieser einzelne Schritt behebt einen großen Teil der „Pop-up-Virus"-Beschwerden.

## Schritt 3: Entfernen Sie verdächtige Erweiterungen

Adware installiert oft eine Browser-Erweiterung. Öffnen Sie in jedem Browser die Erweiterungsseite und **entfernen Sie alles, was Sie nicht bewusst installiert haben** – besonders „Coupon-Finder", „Such-Verbesserer" oder Video-Downloader, an deren Hinzufügen Sie sich nicht erinnern.

- **Chrome / Edge:** Menü → Erweiterungen → Erweiterungen verwalten.
- **Firefox:** Menü → Add-ons und Themes → Erweiterungen.

## Schritt 4: Setzen Sie den Browser zurück

Wenn Pop-ups, eine gekaperte Startseite oder eine geänderte Suchmaschine bestehen bleiben, setzen Sie den Browser auf Standard zurück. Das löscht bösartige Einstellungen und behält dabei Lesezeichen und gespeicherte Passwörter:

- **Chrome:** Einstellungen → Einstellungen zurücksetzen → Einstellungen auf ursprüngliche Standardwerte zurücksetzen.
- **Edge:** Einstellungen → Einstellungen zurücksetzen → Einstellungen auf Standardwerte zurücksetzen.
- **Firefox:** Hilfe → Weitere Informationen zur Fehlerbehebung → Firefox bereinigen.

## Schritt 5: Deinstallieren Sie Adware-Programme

Manche Pop-up-Adware installiert sich als Windows-Programm, nicht nur als Browser-Add-on:

1. **Einstellungen → Apps → Installierte Apps**.
2. Sortieren Sie nach **Installationsdatum** und suchen Sie nach allem, was Sie nicht kennen und das ungefähr zu dem Zeitpunkt installiert wurde, als die Pop-ups begannen.
3. Deinstallieren Sie verdächtige Einträge (gängige Adware-Namen sehen aus wie generische „PC-Optimierer", „Media-Player" oder Toolbars, die Sie nie gewählt haben).

## Schritt 6: Führen Sie einen Malware-Scan durch

Sichern Sie Ihre Entfernung mit einem Scan ab:

1. **Microsoft Defender:** Windows-Sicherheit → Viren- & Bedrohungsschutz → Scanoptionen → **Vollständige Überprüfung**.
2. Für Adware und PUPs speziell ist ein kostenloser On-Demand-Reiniger wie **ADWCleaner** (von Malwarebytes) eigens für Browser-Hijacker und Adware gebaut. Der kostenlose Malwarebytes-Scanner funktioniert ebenfalls.

Stellen Sie alles Gefundene unter Quarantäne und starten Sie neu.

## Schritt 7: Prüfen Sie Autostart und die hosts-Datei (optional)

- **Task-Manager → Autostart-Apps:** unbekannte Einträge deaktivieren, die beim Hochfahren starten.
- **hosts-Datei** (`C:\Windows\System32\drivers\etc\hosts`): manche Adware fügt Weiterleitungseinträge hinzu. Sie sollte normalerweise nur Kommentarzeilen und localhost enthalten; entfernen Sie unbekannte Einträge nur, wenn Sie die Änderung verstehen.

## Schritt 8: Sichern Sie Konten, wenn Sie mit dem Betrug interagiert haben

Wenn Sie ein Passwort oder Zahlungsdaten in ein Pop-up eingegeben oder etwas installiert haben, das es Ihnen aufgetragen hat, behandeln Sie diese Zugangsdaten als kompromittiert. Ändern Sie die betroffenen Passwörter von einem sauberen Gerät aus und aktivieren Sie die Zwei-Faktor-Authentifizierung.

## So vermeiden Sie Pop-up-Viren beim nächsten Mal

- Wenn eine Seite darum bittet, „Benachrichtigungen anzuzeigen", klicken Sie auf **Blockieren**, es sei denn, Sie wollen ihre Updates wirklich.
- Installieren Sie keine Software aus Pop-up-Werbung oder „Ihr Treiber ist veraltet"-Aufforderungen.
- Halten Sie einen Werbe-/Inhaltsblocker und Ihren Browser aktuell.
- Seien Sie skeptisch bei jeder Seite, die Dringlichkeit erzeugt („jetzt handeln, Ihr PC ist infiziert") – diese Dringlichkeit ist der Betrug.

## FAQ

**Sind Pop-ups immer ein Virus?**
Nicht immer. Viele sind Website-Benachrichtigungs-Spam oder Adware statt eines dateiinfizierenden Virus, aber sie sind trotzdem unerwünscht und es lohnt sich, sie mit den obigen Schritten zu entfernen.

**Warum bekomme ich Pop-ups, wenn mein Browser geschlossen ist?**
Das ist meist Browser-Benachrichtigungs-Spam von einer Seite, die Sie versehentlich zugelassen haben. Entziehen Sie ihre Benachrichtigungsberechtigung (Schritt 2), um es zu stoppen.

**Ist das „Microsoft hat einen Virus erkannt, rufen Sie diese Nummer an"-Pop-up echt?**
Nein. Microsoft platziert nie eine Telefonnummer in einem Browser-Pop-up. Es ist ein Tech-Support-Betrug – schließen Sie es und rufen Sie nie an.
